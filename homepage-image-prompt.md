# Humanoid Directory — Homepage Image Generation Prompt

Copy-paste this prompt into ChatGPT, Midjourney, Ideogram, Figma AI, Lovable visual mode, v0, or any design/image AI tool that can create a homepage mockup image.

---

## Prompt

Create a premium homepage design mockup for a website called **Humanoid Directory**.

The website is a global directory and intelligence platform for humanoid robots and the companies building them.

The previous dark version looked futuristic, but it was too black, dense, and slightly scary. The previous light version looked clean, but too generic and boring. Create the ideal middle ground: **futuristic, premium, credible, spacious, and not scary**.

Think of the design as:

- Apple product page clarity
- robotics lab precision
- SpaceX / Tesla cinematic confidence
- Linear / Vercel cleanliness
- Bloomberg Terminal intelligence, but much more approachable
- premium robotics magazine energy

Do **not** make it look like a generic SaaS website. Do **not** make it cyberpunk, military, dystopian, gamer-like, or crypto-like.

---

## Overall Visual Direction

Use a **hybrid light/dark design**:

- Dark cinematic hero section at the top
- Light titanium / off-white body sections below
- Dark premium CTA near the bottom
- Clean, readable cards and tables
- Futuristic details used carefully, not everywhere

The design should feel futuristic through:

- cinematic robot imagery
- technical metadata overlays
- subtle HUD elements
- blueprint/grid details
- thin glowing cyan lines
- premium glass/titanium surfaces
- precise typography
- beautiful robot profile cards

But the page should still feel:

- spacious
- readable
- trustworthy
- modern
- calm
- premium
- not scary

---

## Color Palette

Primary background:
- off-white / titanium white / very light grey for main content
- deep graphite / near-black only for hero and selected premium panels

Accent colors:
- electric blue
- cyan
- titanium silver
- soft graphite
- subtle emerald green for commercial/verified status
- subtle amber/red only for prototype/research badges

Avoid:
- full black page from top to bottom
- excessive neon
- purple crypto gradients
- harsh red warnings
- childish bright colors

The final impression should be: **futuristic robotics, but elegant and inviting**.

---

## Typography

Use premium modern typography:

- Large bold headline
- Sharp but readable sans-serif
- Slightly technical feel
- Use monospace only for small metadata labels, robot IDs, specs, and status tags
- Strong contrast between headings, body text, labels, and specs

Important: text must be large enough to read in the mockup. Avoid tiny unreadable dashboard text.

---

## Layout Requirements

Create a full desktop homepage mockup, **1440px wide**.

The page should look like a finished premium product homepage, not a wireframe.

Use strong section separation, generous spacing, and clear visual hierarchy.

The design should be more spacious than a dense dashboard. Keep the futuristic interface details, but make them controlled and editorial.
